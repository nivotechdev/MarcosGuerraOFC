# **App Name**: Sorriso Premium

## Core Features:

- Hero Section: Prominent headline, subheadline, CTA, and lifestyle image.
- Social Proof Display: Display logos of dental associations and dentist's qualifications.
- Treatment Grid: Showcase dental services (implants, etc.) with minimalist icons and descriptions.
- Clinic Gallery: Display images of the clinic (reception, tech, sterilization) in an asymmetric mosaic layout.
- Location and Contact: Display integrated map (dark mode), address, WhatsApp link, and opening hours.
- Footer Links: Display simplified institutional links and security badges.

## Style Guidelines:

- Primary color: Subtle Sapphire Blue (#0F4C81) for conversion buttons, conveying trust and professionalism.
- Background color: Off-white (#F9F9F9) to create a clean and calming backdrop.
- Accent color: Graphite Gray (#2D2D2D) for text and secondary elements, providing contrast and sophistication.
- Body and headline font: 'Inter' (sans-serif) with varied weights for a clear visual hierarchy. Note: currently only Google Fonts are supported.
- Use generous whitespace (negative space) for a sense of organization and calm.
- Incorporate thin, linear icons to represent dental services in a minimalist style.
- Employ subtle transitions and animations for a smooth and engaging user experience.
